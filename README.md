# Programa de Calificaciones en C++
## Descripción
Este programa fue desarrollado en C++ para registrar y procesar las calificaciones de varios estudiantes. 

El usuario indica cuántos estudiantes va a ingresar (máximo 20), y luego el programa solicita 4 calificaciones por cada estudiante. Después de eso, muestra los resultados en pantalla.

## Qué hace el programa
- Guarda las calificaciones de cada estudiante
- Calcula el promedio individual
- Calcula el promedio general del grupo
- Muestra la nota más alta y la más baja
- Indica cuántos estudiantes aprobaron (70 o más)
- Indica cuántos reprobaron (menos de 70)

## Cómo usé los arreglos
Usé un arreglo bidimensional para guardar las calificaciones, donde cada fila representa un estudiante y cada columna una de sus 4 notas.

También usé otro arreglo para guardar el promedio de cada estudiante.

Los arreglos me permitieron recorrer todos los datos con ciclos y hacer los cálculos más fácil.
